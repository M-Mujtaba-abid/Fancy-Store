E-Commerce RESTful API 
A scalable and high-performance server-side solution for modern e-commerce platforms. This backend is designed to handle product catalogs, user authentication, secure transactions, and order management with a focus on clean architecture.

Core Features
User Management: Secure registration and login using JWT (JSON Web Tokens) and Bcrypt for password hashing.

Product Engine: Full CRUD functionality with support for categories, inventory tracking, and search filtering.

Shopping Cart: Persistent cart logic allowing users to manage items before checkout.

Order Workflow: Comprehensive order processing from initiation to completion.

Secure Payments: Integrated payment gateway support (Stripe/PayPal) for seamless transactions.

Middleware Security: Implementation of CORS, Helmet, and Rate-Limiting to protect against common web vulnerabilities.

Tech Stack
Runtime Environment: Node.js

Web Framework: Express.js

Database: MongoDB (using Mongoose ODM)

Authentication: JWT (State-less Auth)

Documentation: Postman / Swagger (optional)

Getting Started
Prerequisites
Node.js (v16.x or higher)

MongoDB Atlas account or local MongoDB instance
